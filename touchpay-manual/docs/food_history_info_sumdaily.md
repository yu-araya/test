# 社員食堂精算 集計

事業所別に、選択された月の日別実績数が表示されます。
![喫食検索画面 新規登録ボタン](img/food_history_info_sumdaily/food_history_info_sumdaily.png)

| No. | 説明                                                                        |
| --- | --------------------------------------------------------------------------- |
| 1   | 表示したい事業所と対象年月を選択します。                                    |
| 2   | 表示されているデータがCSVで出力出来ます。                                   |
| 3   | 対象の日別の喫食数(※予約メニューに表示されている数量予約数)が表示されます。 |
